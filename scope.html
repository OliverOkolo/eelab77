Troubleshooting

      

        No data / blank waveform

        → Baud rate mismatch — match selector to #define BAUD_RATE in sketch.

        → Close Arduino Serial Monitor — it locks the port.

        → Try a different USB cable (data cable, not charge-only).

        

        High sync error count

        → Almost always a baud rate mismatch. Try 115200 first.

        → Try a slower ADC prescaler (PS_128) in the sketch.

        

        CH2 not showing

        → Enable CH2 in the Channels panel.

        → Confirm ch2Enable = true in your sketch.

        

        Unstable waveform

        → Adjust the trigger level to the midpoint of the signal.

        → Increase holdoff if the signal has high-frequency noise.

        

        Sketch upload fails

        → Click Disconnect first — the web app holds the port.

        

        Firefox not working

        → Web Serial requires Chrome or Edge. Firefox is not supported.
