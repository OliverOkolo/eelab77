Mode Notes

      

        

          DC Voltage

          0–5V direct on A0.

          0–50V with 47k/5.1k divider.

          Accuracy: ±1% (10–90% FS).
        

        

          DC Current

          1Ω shunt on A0.

          Max: 1A continuous.

          Break circuit to insert shunt.
        

        

          Resistance

          Best: 100Ω–100kΩ.

          Disconnect component first.

          10kΩ ref resistor required.
        

        

          Continuity

          Closed when R < 50Ω.

          Same wiring as resistance.

          Visual indicator only (no buzzer).
        

        

          Frequency

          Signal on D8 (0–5V only).

          Range: ~1Hz to ~8MHz.

          Never exceed 5V on D8.
        

        

          Duty Cycle

          Signal on D8 (0–5V only).

          High time / period × 100%.

          Never exceed 5V on D8.
        

        

          Diode Test

          Same wiring as resistance.

          Reads forward voltage Vf.

          OL = open or reverse biased.

Troubleshooting

      

        Reading jumps or is noisy

        → Add 100nF capacitor from A0 to GND.

        → Shorten probe wires to reduce pickup.

        

        Resistance reads OL

        → Check 10kΩ ref resistor is connected 5V→A0.

        → Disconnect component from circuit before measuring.

        

        Frequency reads 0 or OL

        → Signal must be on D8, not A0.

        → Signal must be 0–5V logic. Never exceed 5V.

        → Signal must be at least 1Hz for Timer1 to capture.

        

        No frames received

        → Baud rate mismatch — match selector to sketch.

        → Close Arduino Serial Monitor before connecting.

        

        Firefox not working

        → Web Serial requires Chrome or Edge.
