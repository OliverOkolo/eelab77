Troubleshooting

      

        No data via USB

        → ESP32-S3: check BOARD_ESP32_S3 is defined in sketch.

        → Classic: baud mismatch — match selector to sketch.

        → Close Arduino Serial Monitor before connecting.

        

        Cannot connect via WiFi

        → AP mode: connect your device to EElab77-Scope first.

        → Station mode: confirm the IP address is correct.

        → Check ESP32 LED is blinking — fast = streaming,
               slow = idle.

        

        High latency on WiFi

        → Move device closer to the ESP32.

        → AP mode is more stable than STA mode for scoping.

        → Switch to USB if latency exceeds 80ms.

        

        Waveform noisy or distorted

        → ESP32 ADC has nonlinearity — correction is applied
               automatically in firmware.

        → Add 100nF from GPIO34/35 to GND for RF noise rejection.

        → Never exceed 3.3V on any input pin.

        

        CH2 not showing

        → Enable CH2 in the Channels panel (sidebar).

        → Confirm ch2Enable = true in sketch.

        

        Firefox / Safari not working via USB

        → Web Serial is Chrome/Edge only.

        → Use WiFi mode for full browser support.
